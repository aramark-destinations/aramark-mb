Block Overview
Configure accordion heading and one or more accordion items
Starting point from BA Block Library

Author Experience - Block Configurability
Placement:
Author can add an accordion block to any page section
Accordion Item Dialog Fields
"Summary"
type: text
default: empty
required: yes
field description: "Summary text for the accordion"
"Collapsed by Default"
type: boolean toggle
default: true
required: no
field description: "Whether the panel is collapsed on page load"
"Content"
type: richtext
default: empty
required: no
field description: "Richtext content to display when panel is expanded"
