Block Overview
The Images block should be set up in the EDS codebase with an equivalent Universal Editor component.
Author Experience - Block Configurability
"Section Type" - dropdown
Field description: "Controls the width of the Section and the content inside the Section"
options:
"Full-width" - background expands to full viewport width, internal component area stays constrained to normal max grid width (1320px)
"Contained" - background only expands to max grid width (1320px)
"Full-Bleed" - background expands to full viewport width, internal component area allowed to expand to viewport width - 60px left & right
"Background" (just a heading in the dialog for this group of fields)
"Image" - path to DAM
"Alt" - text field
"Color" - click-select color swatches: Dark (secondary 500), Light (white), Light 2 (gray 50), Brand (primary 500), Tertiary (tertiary 500)
"Opacity Overlays" (applied over background)
Full Overlay - dropdown - options:
None
Darken - 20%
Darken - 40%
Lighten - 20%
Lighten - 40%
Linear Gradient - dropdown - options:
None
Bottom to Top
Top to Bottom
Left to Right
Right to Left
Starting Opacity - number 0-100 (explore configurable opacity/color)
Ending Opacity - number 0-100
"Section Theme Style" - dropdown
Field description: "Controls color styling applied to components in this Section"
options:
Dark
Light
Light2
Brand
Tertiary
