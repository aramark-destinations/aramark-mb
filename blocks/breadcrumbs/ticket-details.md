Block Overview
The Breadcrumb block should be set up in the EDS codebase
The Breadcrumb is dynamically constructed based on site structure above the current page
"Home" is always the first link/element and the rest of the breadcrumb is determined from the site structure
text value displayed for each page is from "Breadcrumb Title" page metadata field
if empty, fallback to use page "Title" value
Author Experience - Block Configurability
Placement/Usage
Not available to add to pages on its own
Author will be able to include Breadcrumb via the Hero Block
Dialog Fields
no configurations
End User Experience - Style & Functionality
Breadcrumb displays in the bottom left corner of the Hero when an author has included it
The first link in the breadcrumb is always "Home" linked to the property homepage/index
